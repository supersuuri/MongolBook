# Demo видео оруулах заавар

1. `docs/` хавтас руу өөрийн видео файлыг хийгээрэй (жишээ: `demo.mp4`).
2. Видео форматыг MP4 (H.264) болгохыг зөвлөж байна.
3. README-д `docs/demo.mp4` руу холбоос хийх эсвэл frontend-д холбогдоно.

Жич: Видео их том бол Git LFS эсвэл гадаад хост (YouTube, Vimeo) ашиглан холбохыг зөвлөж байна.
